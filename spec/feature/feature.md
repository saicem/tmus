# 功能规范

## 概览

展示一些常见的统计项，具体如下：

1. 今日使用的应用数量
2. 今日使用的总时长
3. 年度每日使用时长的热力图
4. 周使用时长的热力图（和上周对比）
5. 今日使用应用的详情
6. 日内节律（今日对比最近一周的使用时间在一天内的分布）

## 时间线

按日倒序展示每日使用的应用，每日的应用按时长倒序展示。
